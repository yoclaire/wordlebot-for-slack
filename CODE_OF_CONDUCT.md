# Code of Conduct

## The short version

Be kind. Be respectful. Assume good intent.

## The longer version

This project is open to everyone regardless of experience level, background, or identity. We're here to make a fun Wordle bot, not to gatekeep.

**Expected behavior:**
- Be welcoming and inclusive
- Give and receive constructive feedback gracefully
- Focus on what's best for the project

**Unacceptable behavior:**
- Harassment, insults, or personal attacks
- Trolling or deliberately inflammatory comments
- Publishing others' private information

## Enforcement

If someone is making the project unwelcoming, open an issue or contact the maintainer directly. Reports will be handled privately.

Participants who violate this code of conduct may be removed from the project at the maintainer's discretion.
